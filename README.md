# Moodle-XML2TeX
Scripts for converting between MoodleXML question format and LaTeX

## Usage

### xml2tex.py

`xml2tex.py <filename> [-F]`

-F	overwrites the output file in case it exits

### tex2xml.py

`tex2xml.py <filename> [-e] [-F]`

-e	uses English instead of Croatian feedback

-F	overwrites the output file in case it exits
